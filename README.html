💡 使い方

📱 推奨：オンボーディングから体験
08_welcome.html から始めると、実際のアプリのような連続した体験ができます。
  1. ウェルカム画面から開始（4スライドをスワイプ）
  2. アカウント登録で基本情報を入力
  3. グループ作成でグループタイプを選択
  4. AIトレーナー設定でカスタマイズ
  5. グループタイムラインでメイン機能を体験

🎯 各機能の試し方

  • 投稿を見る: タイムラインの投稿カードをタップ
  • 投稿を作成: ボトムナビの中央「+」ボタン
  • 進捗を確認: ボトムナビ「進捗」タブ
  • AIに相談: ボトムナビ「AI相談」タブ
  • 設定変更: ボトムナビ「設定」タブ
  • プロフィール: タイムライン右上のアイコンまたは設定画面から
